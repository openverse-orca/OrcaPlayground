## 键盘控制

### 策略控制
| 按键 | 功能 |
|------|------|
| `]` | 启用策略控制（站立） |
| `o` | 停止策略控制 |
| `i` | 重置到初始状态 |
| `=` | 切换站立/行走模式 |

### Mimic 动作
| 按键 | 功能 |
|------|------|
| `[` | 执行/取消 Mimic 动作 |
| `;` | 切换到下一个 Mimic 动作 |
| `'` | 切换到上一个 Mimic 动作 |

### 移动控制（行走模式下）
| 按键 | 功能 |
|------|------|
| `w/s` | 前进/后退 |
| `a/d` | 左移/右移 |
| `q/e` | 左转/右转 |
| `z` | 停止移动 |

### 其他
| 按键 | 功能 |
|------|------|
| `1/2` | 增加/减少站立高度 |
| `4/5/6/7` | 调整 Kp 增益 |
| `0` | 重置 Kp 增益 |
